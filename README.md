🚀 Biroor Block Coding Studio (Lab)
Biroor Lab is an interactive, web-based visual programming platform designed to bridge the gap between block-based logic and professional syntax. It provides a "Scratch-like" experience while giving students a real-time look at the code they are creating.

✨ Key Features
Visual Block Palette: Familiar categories including Motion, Looks, Sound, Events, and Control, making it easy for beginners to start coding instantly.

Real-time JavaScript Generation: As users snap blocks together, the editor automatically generates clean, readable JavaScript code in the side panel. This helps students understand the logic behind the syntax.

Interactive Stage & Assets: A built-in canvas for sprites and backdrops, featuring a "Green Flag" execution system for immediate feedback.

Project Management: Full support for saving and loading projects, including SB3 (Scratch) Import/Export compatibility to ensure a seamless transition for students coming from other platforms.

Built-in Debugger & Console: A professional console output to track program execution and debug logic errors in real-time.

🛠 Tech Stack
Frontend: Modern Web Technologies (React/Vue or Vanilla JS based on the UI)

Logic: Custom block-to-code transpiler

Compatibility: Fully responsive and optimized for school environments (Smart Boards/e-Tahta).

🎯 Why Biroor Lab?
Most block-coding tools keep the "magic" hidden. Biroor Lab makes the magic transparent by showing the code, empowering the next generation of software engineers to move from visual dragging to professional typing.
